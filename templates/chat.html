Help & Tips

To get the best results, try these question formats:

  • Specific questions: "What does the document say about climate change impacts?"
  • Comparative questions: "Compare the different approaches to renewable energy mentioned in the document."
  • Summary requests: "Summarize the key findings about urban development."
  • Evidence questions: "What evidence supports the claim that exercise improves mental health?"

For best results with document uploads:

  • Multiple files: You can upload multiple documents at once by holding Ctrl/Cmd while selecting files.
  • Supported formats: PDF, JPG, JPEG, and PNG files are supported.
  • Text quality: Documents with clear, searchable text work best. Scanned documents may have variable results.
  • File size: Larger files may take longer to process.

Themes: These are key topics or concepts identified in your documents. They help you understand the main subjects discussed.

Knowledge Graph: This visualization shows how documents and themes are connected:

  • Blue nodes: Document sections
  • Green nodes: Identified themes
  • Connections: Show relationships between documents and themes
  • Highlighted nodes: Most relevant to your current query

You can interact with the graph by zooming, panning, and clicking on nodes for more information.

Common issues and solutions:

  • No results: Try rephrasing your question to be more specific.
  • Processing errors: If document processing fails, try a different file format or a clearer document.
  • Slow response: Large documents may take longer to process. Please be patient.

Session Reset Information

This application is designed to reset completely when you refresh or reload the page.

Refreshing the page will:

  • Create a new clean session
  • Delete all previously uploaded documents
  • Clear all chat history
  • Reset all knowledge graphs and themes

This design ensures:

  • Complete isolation between research sessions
  • Optimal memory usage
  • Better performance
  • Clean state for each new research session
Tip: If you want to start fresh at any time, simply refresh the page!
